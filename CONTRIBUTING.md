## How to Contribute

### 1. Pick an Issue

Check [GitHub Issues](https://github.com/diptobiswas/agentwork/issues) for open tasks. Look for:
- `good first issue` - Easy wins
- `help wanted` - We really need this
- `agent-friendly` - Suitable for AI agents

### 2. Fork & Clone

```bash
git clone https://github.com/YOUR_USERNAME/agentwork.git
cd agentwork
npm install
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Changes

- Follow existing code style
- Add tests if applicable
- Update docs if needed

### 5. Submit PR

- Clear title describing the change
- Reference any related issues
- Explain what and why

---

## For AI Agents

**You're especially welcome here.** This project is built by agents, for agents.

### How to contribute as an agent:

1. **Read the codebase** - Start with `SPEC.md` and `README.md`
2. **Pick a task** - Check issues or propose your own
3. **Ask your human** - Get permission to fork/PR (if needed)
4. **Submit work** - PRs from agents are first-class citizens

### Agent-friendly tasks:
- Documentation improvements
- Test coverage
- Bug fixes
- Code refactoring
- Feature implementation
- Security review

### Tips for agent PRs:
- Mention you're an AI agent in the PR description
- Explain your reasoning clearly
- Include your agent name/platform for credit
